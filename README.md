# Voting Eligibility Checker

A simple Java application that determines whether a person is eligible to vote based on their age.

## Project Description

This is a desktop application built with Java/Swing that checks voting eligibility. The application prompts the user to enter their age and determines if they meet the minimum age requirement (18 years old) to vote.

## Features

- Simple user input interface using Swing GUI
- Age validation for voting eligibility
- Clear feedback on voting eligibility status

## Requirements

- Java Development Kit (JDK) 8 or higher
- NetBeans IDE (optional, for development)

## How to Build

Using Ant:
```bash
ant clean build
```

## How to Run

Using Ant:
```bash
ant run
```

Or run the compiled JAR:
```bash
java -jar dist/votingEligibility.jar
```

## Project Structure

- `src/votingeligibility/` - Source code files
  - `VotingEligibility.java` - Main application logic
  - `NewJFrame.java` - Swing GUI components
- `build/` - Build output directory
- `nbproject/` - NetBeans project configuration
